## 🚀 启动和使用

### 1. 启动开发服务器
```bash
cd ice-king-ai-next
npm run dev
```

### 2. 配置环境变量
确保 `.env.local` 中包含：
```env
OPENAI_API_KEY=your_api_key_here
OPENAI_BASE_URL=https://api.openai.com/v1
```

### 3. 测试API调用
1. 访问 http://localhost:3000/simple-test.html
2. 按F12打开控制台
3. 点击测试按钮
4. 观察详细的API调用日志

### 4. 体验完整功能
1. 访问主页查看动效
2. 点击"开启内容创作之旅"
3. 体验完整的内容生成流程
