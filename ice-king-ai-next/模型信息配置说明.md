# 模型信息 Badge 配置说明

## 概述
首页现在显示基于 DeepSeek-V3.2-Speciale 模型的实现信息，这个信息以可配置的 badge 形式展示。

## 位置
Badge 位于首页的 hero section 中，在主要的"从灵感乍现，到爆款刷屏"标签下方。

## 配置方式

### 1. 修改模型名称
编辑文件：`src/components/ModelInfoBadge.tsx`
```typescript
// 修改默认模型名称
const modelName = 'Your-New-Model-Name';
```

### 2. 修改显示文本
```typescript
// 自定义显示文本
<ModelInfoBadge 
  displayText="基于最新的 GPT-4 模型实现" 
/>
```

### 3. 完全隐藏 Badge
```typescript
// 隐藏整个badge
<ModelInfoBadge showBadge={false} />
```

### 4. 自定义样式
```typescript
// 添加自定义样式
<ModelInfoBadge 
  className="custom-class-name"
  modelName="Custom Model"
  displayText="自定义文本"
  showBadge={true}
/>
```

## 完整示例

### 更改模型信息
```typescript
// 示例：更改为其他模型
<ModelInfoBadge 
  modelName="GPT-4-Turbo"
  displayText="基于最新的 GPT-4-Turbo 模型实现"
/>
```

### 完全移除
```typescript
// 方法1：直接移除组件
// 删除 src/app/page.tsx 中的 <ModelInfoBadge />

// 方法2：设置为不显示
<ModelInfoBadge showBadge={false} />
```

## 样式特点
- 🎨 **蓝色主题**: 使用蓝色配色方案，与主要yellow主题形成层次
- ⚡ **动画效果**: 包含脉冲动画和CPU图标
- 📱 **响应式**: 自动适应不同屏幕尺寸
- 🎯 **适度突出**: 不会抢夺主要内容的注意力
