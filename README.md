## Getting Started

### Prerequisites
- Node.js 18+ and pnpm
- Modern web browser with ES2020 support

### Installation
```bash
# Clone the repository
git clone <repository-url>
cd ice-king-ai-viral-content-machine

# Install dependencies
pnpm install

# Start development server
pnpm dev

# Build for production
pnpm build
```

### Usage
1. **Input Your Details**: Fill out the content generation form
2. **Generate Content**: Click the lightning bolt button to create your viral content package
3. **Review & Copy**: Browse platform-specific content and copy what you need
4. **Optimize**: Use quality scores and A/B headlines to maximize performance
